What is Automated ML ?

Automated machine learning, also referred to as automated ML or AutoML, is the process of automating the time-consuming, iterative tasks of machine learning model development. It allows data scientists, analysts, and developers to build ML models with high scale, efficiency, and productivity all while sustaining model quality.

Guidelines To Use Automated ML

  1. Paste the Github URL of the dataset.
    ex: https://raw.githubusercontent.com/rohan-r15/Datasets/main/entreprener.csv
  2. The target column must be the last column in the dataset.
  3. You can paste the URL taken from any of the Github repositories.
  4. Select the type of problem you are looking to solve either Classification or Regression.
  5. Now click the 'Get Model' button.
  6. Now your model is being built, It takes some time(depends upon the dataset), so wait until you get the best model name, best model parameters, and best features to select from the dataset.

Consider executing few times to get the best model.

It takes some time to build the best model so please wait until the model is ready.
